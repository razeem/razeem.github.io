03 Projects

Qatar University

Current

Drupal 11 platform for Qatar's national university. Technical Lead: component-based theming with Single Directory Components on Bootstrap 5, config-sync driven delivery, Playwright QA automation, GrumPHP / PHPStan quality gates and Dockerised environments deployed via Azure Pipelines.

Engineered, architected and co-authored 20+ bespoke modules for the platform — from AI content indexing and visual editing tools to editorial workflow and access control.

Explore the custom modules
  • Generic AI Indexer — high-performance JSON API that transforms nodes, paragraphs and media into clean, view-mode-driven structured data for AI ingestion — resolving translations, revisions and nested content with recursion-safe rendering.
  • Universal Header Resolver — lets each college or section present its own site header: config entities bind a menu and a config page to weighted trigger conditions (college taxonomy, content type), with optional per-node overrides.
  • Image Marker Paragraphs — click-and-drag visual widget to place hotspot markers on an image at percentage positions, each marker rendering any paragraph as its detail content; ships its own paragraph types and SDC components, and degrades to manual X/Y entry without JS.
  • Paragraph Revision Compare — paragraph-aware diff between two node revisions for content approvers: classifies each paragraph as added, removed or changed, pairing by stable ID and recursing into nested paragraph structures.
  • User Access Scope — maps users to content scopes (taxonomy-based or flat realm) and enforces visibility, editing and creation through Drupal's node-access grants; emits review-workflow notifications on moderation transitions.
  • Notifications — CMS notification subsystem recording content-state changes (submitted / approved / rejected) and surfacing them in a top-bar bell with live AJAX counts, plus per-user cron cleanup.
  • University Org Mapper — org-node entity with parent / weight hierarchies, a fast tree-editing UI, and a custom field type so any entity can embed an organisational chart.
  • Advanced Menu Breadcrumb — centralises all breadcrumb logic into one UI-configurable module supporting menu-trail and custom per-content-type strategies.
  • Inner Page Menu — per-page navigation menus managed inline on the content edit form: each page silently owns its own auto-provisioned menu with drag-and-drop hierarchy.
  • Migration Utilities — a toolbox of ~15 reusable Migrate API source / process plugins covering HTML extraction, value shaping, reference building and file handling.
